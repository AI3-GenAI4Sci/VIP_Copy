# Workflow Skills

`current/` is the active skill-loading surface. Keep only skills that the
runtime should load for the default workflow here.

For personalized copy generation, the active path is the merged
`current/personalized-copy-generation/SKILL.md` session. It mines factors and
writes copy in one shared context while preserving separate factor and copy
artifacts.

`archive/` keeps retired skill prose for reference. Archived skills are readable
history, not production load targets.
