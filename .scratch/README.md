# Scratch

Use this directory for local issue drafts, PRDs, and temporary notes for active
GSD work.

Create one folder per feature or investigation:

```text
.scratch/<feature-slug>/
```

Do not store generated caches, provider traces, or long-term documentation here.
Promote durable conclusions into `.planning/`, `meta/memory.md`,
`meta/rubrics.md`, or `docs/`.
